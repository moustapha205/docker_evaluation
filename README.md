# Projet Déploiement Local Docker - TD

Ce projet démontre la conteneurisation d'une application locale en utilisant Docker et Docker Compose.

##Comment lancer le projet
1. Assurez-vous d'avoir **Docker Desktop** installé.
2. Clonez le dépôt.
3. Dans le terminal, à la racine du projet, lancez :
   ```bash
   docker-compose up -d --build
